# Capstone 3: Business Analysis with Power BI

## Project Overview

This project analyzes sales performance data for **EmporiUm**, a student bookstore chain. The dashboard focuses on the **South Region** under **Len Jensen**, providing insights into sales trends, store performance, customer activity, and product category performance.

## Objectives

* Analyze monthly sales trends in the South Region
* Evaluate store performance
* Identify top customers
* Analyze product category sales
* Provide data-driven business recommendations

## Tools Used

* Power BI Desktop
* Power Query
* DAX

## Dashboard Highlights

* Monthly Sales Trend
* Store Sales Performance
* Sales by Product Category
* Top Customers
* KPI Cards for Total Sales, Transactions, and Average Sale
* Interactive slicers for Store, Product Category, and Month

## Key Takeaways

The dashboard provides insights into sales performance across the South Region managed by **Len Jensen**, helping identify top-performing stores, customers, and product categories to support informed business decisions.
